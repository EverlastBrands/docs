# GET Public Base Products

Retrieves all base products

| verb      | path                  |
| --------- | --------------------- |
| **`GET`** | /base-products/public |

## Example

**Response**

```json
{
  "data": {
    "baseProducts": [
      {
        "description": "",
        "order": 0,
        "childProducts": ["7nwqNO2oIr9LyMGX5y9n", "M07IXoWR0prJ07kBmEhy"],
        "categoryId": "mmXLzSSMRjDNiibXeh3t",
        "name": "Drip Edge",
        "id": "HBaR6rEAayR89lXJ11iY"
      },
      {...}
    ]
  }
}
```
